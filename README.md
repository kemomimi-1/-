# 🧠 EEG-AI 实时脑电分析平台 v3.0

这是一个集成了实时脑电数据获取、AI 智能分析及可视化弹幕交互的综合平台。

## 🚀 团队快速启动指南

### 1. 环境准备 (拉取代码后必做)
由于 GitHub 文件大小限制，仓库中不包含以下大文件，请从团队**内部分享链接**下载并放置在根目录下：
- [ ] `InfluxDB Core 3.2.1/` (核心数据库)
- [ ] `openbcigui_v6.0.0-beta.1_windows64/` (数据采集工具)

### 2. 软件要求
- **Java SDK**: 推荐使用 JDK 21 (已于 `start_all_services.bat` 中配置)
- **Maven**: 用于后端依赖构建

### 3. 一键运行 ⚡
在根目录下找到并双击运行：
👉 **`start_all_services.bat`**

该脚本会自动执行：
1. 启动本地 InfluxDB 数据库。
2. 启动 Spring Boot 后端服务。
3. 倒计时 30 秒后自动打开浏览器访问：`http://localhost:8080/index.html`

---

## 🔥 核心优化点 (v3.0 更新)
- [x] **Premium 级 UI**: 提示框重构为 Glassmorphism 磨砂玻璃质感。
- [x] **硬核归零逻辑**: 页面刷新即自动同步断开所有背景连接，保障系统稳定性。
- [x] **异步熔断器**: 解决分析启动期间断开连接导致的竞态冲突问题。
- [x] **时间降噪**: AI 诊断中移除冗余时间戳，并同步为北京本地时间显示。

---

## 🛠️ 项目结构
- `ai_eeg-main/`: 后端源码与前端 UI 文件。
- `start_all_services.bat`: 一键启动所有服务。
- `.gitignore`: 已优化，排除大型二进制文件。

## ⚠️ 注意事项
- 本地机器需预留 `8080` 端口。
- 首次运行建议先检查本地 `JAVA_HOME` 路径是否与 `.bat` 文件中配置的一致。
