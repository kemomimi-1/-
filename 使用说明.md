# 🧠 EEG-AI 分析平台队员协作指南

本指南旨在帮助队员从零开始配置环境并成功运行本项目。本项目是一个基于 Spring Boot + InfluxDB + 阿里云通义千问大模型的脑电波实时分析平台。

## 1. 软件环境要求
在开始之前，请确保你的电脑安装了以下环境。**如果以下软件你都没有安装过，请务必按照步骤来：**
*   **操作系统**: Windows 10 或 Windows 11
*   **JDK 21**: (必须是 21，因为 Spring Boot 版本要求)。可以去 Oracle 官网或者清华镜像源下载安装。安装后配置环境变量。
*   **MySQL 8.0+**: 用于存储用户信息和分析历史。推荐直接下载安装包，记住你设置的 root 密码。
*   **InfluxDB Core 3.2.1**: (用于存储高频、海量的时序脑电波数据)。
*   **OpenBCI.GUI**: (用于运行脑电波数据模拟器)。
*   **IDE 开发工具**: 强烈推荐最新版的 IntelliJ IDEA (社区版即可)。

---

## 2. 数据库配置

### A. MySQL 设置
1.  启动你的 MySQL 服务。
2.  使用你喜欢的数据库工具（如 Navicat, DataGrip 或直接用自带的命令行）登录 MySQL。
3.  创建一个和项目对应的数据库并设置字符集：
    ```sql
    CREATE DATABASE eeg_platform DEFAULT CHARACTER SET utf8mb4;
    ```
4.  执行项目源码中提供的 `eeg_platform.sql` 文件（如果有的话），或者如果你知道 Hibernate 会自动建表，可以略过此步。
5.  **关键步骤**：打开 IDEA，找到项目里的 `src/main/resources/application.yml` 文件。找到 `datasource` 配置部分，把 `password: 2745099813Aa` 这里改成你**自己电脑上 MySQL 的 root 密码**。

### B. InfluxDB 设置
1.  **启动 InfluxDB**：
    *   如果你是下载的 Windows 压缩包，解压后双击运行里面的 `influxdb3.exe` 文件。会出现一个黑框框，**不要关闭它**，说明数据库正在后台运行。
    *   （如果有防火墙提示，请点击“允许访问”）。
2.  确保 InfluxDB 运行在默认端口 `8181` 上（黑框里如果没有报错就是正常的）。
3.  **注意**：目前项目所需的 API Token、Bucket 和 Org 已经内置在代码里了（`application.yml` 的 `influxdb.token`），你只需要把 `influxdb3.exe` 跑起来即可，**不需要**打开浏览器额外去配置。

## 3. 使用 OpenBCI GUI 发送脑电数据
本项目默认通过 UDP 协议接收脑电数据报文。请使用官方自带的 OpenBCI GUI 进行数据转发：
1.  **打开 OpenBCI GUI**，选择你的数据源（页面左上角选择Synthetic 模拟数据）然后点击 `START SESSION`。
2.  在右下角的下拉选择框（Accelerometer）点击，在弹出的菜单里选择 **Networking**。
3.  在 Networking 面板中，我们将配置 UDP 数据流，对应后端页面分配给你的 3 个端口（比如 `15001`, `15002`, `15003`）：
    *   **Protocol**: 选择 `UDP`
    *   **IP Address**: `127.0.0.1` 
    *   **Port**: 根据页面的控制连接的Raw，Filt,Band依次填入
    *   **Data Type**: 选择你要传出的数据类型，第一个15001选择TimeSeriesRaw，第二个15002选择TimeSeriesFilt，第三个15003选择BandPower
4.  点击 下方 Start UDP Stream和左上角Start Data Stream即可
---

## 4. 启动后端项目
1.  打开 IntelliJ IDEA。
2.  选择 `File` -> `Open`，找到并选中 `ai_eeg-main\ai_eeg-main\source\eeg-fileData` 这个目录并打开。
3.   IDEA 会在右下角提示你加载 Maven 项目，点击同意即可，耐心等待依赖自动下载完成（可能需要几分钟）。
4.  在左侧的项目结构树中，找到启动类：`src/main/java/com/eeg/EegFileDataApplication.java`。
5.  右键点击这个文件，选择 **Run 'EegFileDataApplication'**。

---

## 5. 前端测试与功能展示
后端跑起来之后，打开你的谷歌浏览器 (Chrome) 或者 Edge：
1.  **访问项目首页**：在地址栏输入 `http://localhost:8080/index.html`。
2.  **账号登录**：如果有 SQL 数据可以直接用已有账号，或者随便注册一个测试账号登录。
3.  **连接与启动**：
    *   点击左边侧边栏的 **“需求连接”**，界面会分发给当前用户 3 个端口。
    *   点击上方中间的 **“▶ 播放（启动分析）”** 按钮。
    *   点击网页右边的悬浮工具条里的 **“查看弹幕历史”**。
4.  **结果展示**：如果你已经按照第 3 步通过 OpenBCI GUI 开启了 Networking 的 UDP 转发并 Start，在页面右侧数据流状态点击展开可以看到有数据
