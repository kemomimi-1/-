# 项目启动

## 1.先启动 InfluxDB3 Core 3.2.1 数据库服务

![image-20251018165750722](C:\Users\mrjzh\Desktop\关于AI+MCP联动脑电数据相关项目&文档\项目运行指南\pic\InfluxDB数据库.png)

​	

```bash
# 在当前文件夹输入cmd打开命令行窗口

# 输入数据库服务启动命令
influxdb3.exe serve --node-id mynode --object-store file --data-dir ./influxdb-data

#命令行解析
influxdb3.exe serve ^
  --node-id mynode            ← 给节点起个名字，集群里必须唯一  
  --object-store file         ← 明确把对象存储落在“本地文件系统”  
  --data-dir ./influxdb-data  ← 指定数据根目录（会自动生成子目录 wal/、parquet/ 等）
```
