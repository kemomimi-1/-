# 脑电数据分析系统 - 对话历史功能说明文档

## 1. 系统概述

本模块是基于Spring Boot框架构建的对话历史管理系统，为脑电数据分析平台提供类似主流AI产品（如ChatGPT）的对话会话体验。系统支持用户与AI之间的多轮对话记录、会话管理、收藏功能和统计分析，结合MCP工具调用记录，为用户提供完整的AI交互历史追踪。

## 2. 系统架构组件

### 2.1 控制层（ConversationHistoryController.java）

**类职责：** 处理对话历史相关的HTTP请求，提供完整的RESTful API接口，聚焦核心对话会话管理功能。

#### 核心接口功能：

**`POST /api/conversations/new`** - 新建对话会话

- **功能描述：** 创建新的对话会话，为用户开始新的AI对话提供独立的会话空间
- **权限验证：** 基于HttpSession验证用户登录状态
- **会话生成：** 调用Service层生成唯一会话ID
- **使用意义：** 提供类似ChatGPT的对话会话隔离，每个会话包含独立的对话上下文

```java
@PostMapping("/new")
public ResponseEntity<Object> createNewConversation(HttpSession httpSession) {
    Long userId = (Long) httpSession.getAttribute("userId");
    if (userId == null) {
        return ResponseEntity.status(401).body(createErrorResponse("用户未登录"));
    }

    try {
        log.info("用户 {} 创建新对话会话", userId);

        ConversationSession newSession = conversationHistoryService.createNewConversationSession(userId);

        Map<String, Object> responseData = new HashMap<>();
        responseData.put("session", newSession);
        responseData.put("message", "新对话会话已创建");

        return ResponseEntity.ok(createSuccessResponse(responseData));

    } catch (Exception e) {
        log.error("创建新对话会话失败 - 用户: {}", userId, e);
        return ResponseEntity.badRequest().body(createErrorResponse("创建新对话失败: " + e.getMessage()));
    }
}
```

**`GET /api/conversations`** - 获取用户对话会话列表

- **功能描述：** 分页查询用户的所有对话会话，按最后更新时间倒序排列
- **分页参数：** 支持page（页码）和size（页大小）参数，默认每页20条
- **数据优化：** 只返回每个会话的最新消息作为会话代表
- **使用意义：** 为前端提供会话列表展示，支持大量历史会话的高效浏览

```java
@GetMapping
public ResponseEntity<Object> getConversationSessions(
        @RequestParam(defaultValue = "0") int page,
        @RequestParam(defaultValue = "20") int size,
        HttpSession httpSession) {

    Long userId = (Long) httpSession.getAttribute("userId");
    if (userId == null) {
        return ResponseEntity.status(401).body(createErrorResponse("用户未登录"));
    }

    // 参数验证
    if (page < 0) page = 0;
    if (size <= 0 || size > 100) size = 20;

    try {
        Page<ConversationSession> sessions = conversationHistoryService
                .getUserConversationSessions(userId, page, size);

        Map<String, Object> responseData = new HashMap<>();
        responseData.put("sessions", sessions.getContent());
        responseData.put("pagination", createPaginationInfo(sessions));
        responseData.put("totalSessions", sessions.getTotalElements());

        return ResponseEntity.ok(createSuccessResponse(responseData));
    }
}
```

**`GET /api/conversations/{sessionId}/messages`** - 获取会话消息记录

- **功能描述：** 获取特定会话的所有对话消息，按时间正序排列
- **权限控制：** 验证会话所有权，确保用户只能访问自己的对话记录
- **数据完整性：** 返回完整的用户查询和AI回复内容
- **会话信息：** 附带会话基本信息（标题、创建时间、收藏状态）
- **使用意义：** 为前端提供完整的对话历史展示，支持用户回顾完整对话内容

```java
@GetMapping("/{sessionId}/messages")
public ResponseEntity<Object> getSessionMessages(
        @PathVariable String sessionId,
        HttpSession httpSession) {

    Long userId = (Long) httpSession.getAttribute("userId");
    if (userId == null) {
        return ResponseEntity.status(401).body(createErrorResponse("用户未登录"));
    }

    try {
        List<ConversationHistory> messages = conversationHistoryService.getSessionMessages(userId, sessionId);

        Map<String, Object> responseData = new HashMap<>();
        responseData.put("sessionId", sessionId);
        responseData.put("messages", messages);
        responseData.put("messageCount", messages.size());

        // 添加会话基本信息
        if (!messages.isEmpty()) {
            ConversationHistory firstMessage = messages.get(0);
            ConversationHistory lastMessage = messages.get(messages.size() - 1);

            responseData.put("sessionInfo", Map.of(
                    "title", firstMessage.getSessionTitle() != null ?
                            firstMessage.getSessionTitle() : firstMessage.generateDefaultTitle(),
                    "createdAt", firstMessage.getCreatedAt(),
                    "updatedAt", lastMessage.getUpdatedAt(),
                    "isBookmarked", lastMessage.isBookmarked()
            ));
        }

        return ResponseEntity.ok(createSuccessResponse(responseData));
    }
}
```

**`generateDefaultTitle()`** - 自动生成会话标题

- **智能截断：** 基于第一条用户查询生成合适长度的标题
- **断点优化：** 在合理位置截断，避免截断单词中间
- **使用意义：** 为用户提供直观的会话标识，无需手动命名

```java
public String generateDefaultTitle() {
    if (userQuery == null) return "新对话";

    String query = userQuery.trim();
    if (query.length() <= 30) {
        return query;
    }

    // 尝试在30字符内找到合适的断点
    int cutPoint = 27;
    while (cutPoint > 20 && !Character.isWhitespace(query.charAt(cutPoint))) {
        cutPoint--;
    }

    if (cutPoint <= 20) {
        cutPoint = 27; // 如果没找到合适断点，就强制截断
    }

    return query.substring(0, cutPoint) + "...";
}
```

**`saveConversationToSession()`** - 保存对话到会话

- **完整记录：** 保存用户查询、AI回复、工具使用、性能数据等完整信息
- **自动分类：** 调用智能分类功能自动识别查询类型
- **标题生成：** 为新会话自动生成基于首个查询的标题
- **权限验证：** 确保只能向用户自己的会话添加消息
- **使用意义：** 为AI查询控制器提供统一的对话存储接口

```java
@Transactional
public ConversationHistory saveConversationToSession(String sessionId, Long userId, String userQuery,
                                                     String aiResponse, Long eegSessionId,
                                                     Object responseTokens, List<String> toolsUsed,
                                                     Long processingDurationMs) {
    try {
        // 验证会话权限
        if (!isSessionOwnedByUser(sessionId, userId)) {
            throw new RuntimeException("无权访问该对话会话");
        }

        ConversationHistory conversation = new ConversationHistory();
        conversation.setUserId(userId);
        conversation.setConversationSessionId(sessionId);
        conversation.setUserQuery(userQuery);
        conversation.setAiResponse(aiResponse);
        conversation.setConversationTimestamp(LocalDateTime.now());
        conversation.setEegSessionId(eegSessionId);
        conversation.setProcessingDurationMs(processingDurationMs);

        // 设置工具使用信息
        boolean usedTools = toolsUsed != null && !toolsUsed.isEmpty();
        conversation.setUsedMcpTools(usedTools);
        if (usedTools) {
            conversation.setToolsUsed(objectMapper.writeValueAsString(toolsUsed));
        }

        // 自动分类查询类型
        ConversationHistory.QueryCategory category = ConversationHistory.QueryCategory.categorizeQuery(userQuery);
        conversation.setQueryCategory(category.name());

        // 如果是会话的第一条消息，生成会话标题
        List<ConversationHistory> existingMessages = conversationRepository.findSessionMessages(userId, sessionId);
        if (existingMessages.isEmpty()) {
            String title = conversation.generateDefaultTitle();
            conversation.setSessionTitle(title);
        }

        return conversationRepository.save(conversation);
    }
}
```

**`toggleSessionBookmark()`** - 切换会话收藏状态

- **状态检测：** 获取会话最新消息以确定当前收藏状态
- **批量更新：** 更新整个会话的所有消息收藏状态
- **状态返回：** 返回更新后的收藏状态供前端同步
- **使用意义：** 提供直观的收藏功能，帮助用户管理重要对话

```java
@Transactional
public boolean toggleSessionBookmark(Long userId, String sessionId) {
    try {
        // 获取会话的最新消息以确定当前收藏状态
        List<ConversationHistory> messages = conversationRepository
                .findLastMessageInSession(userId, sessionId, PageRequest.of(0, 1));

        if (messages.isEmpty()) {
            throw new RuntimeException("会话不存在");
        }

        boolean currentBookmarked = messages.get(0).isBookmarked();
        boolean newBookmarkStatus = !currentBookmarked;

        // 更新整个会话的收藏状态
        conversationRepository.updateSessionBookmarkStatus(userId, sessionId, newBookmarkStatus);

        return newBookmarkStatus;
    }
}
```

**`isSessionOwnedByUser()`** - 会话权限验证

- **数据库检查：** 首先检查数据库中是否存在该会话记录
- **新会话验证：** 对于新创建的会话，通过ID格式验证权限
- **时间验证：** 验证新会话的时间戳在合理范围内
- **使用意义：** 确保数据安全，防止用户访问他人的对话记录

```java
private boolean isSessionOwnedByUser(String sessionId, Long userId) {
    if (sessionId == null || userId == null) {
        return false;
    }

    try {
        // 首先检查数据库中是否已存在该会话的记录
        boolean existsInDb = conversationRepository.existsByConversationSessionIdAndUserId(sessionId, userId);

        if (existsInDb) {
            return true;
        }

        // 如果数据库中不存在，检查是否是新创建的会话ID
        // 新会话ID格式：user_{userId}_{timestamp}
        if (sessionId.startsWith("user_" + userId + "_")) {
            try {
                String timestampPart = sessionId.substring(("user_" + userId + "_").length());
                long timestamp = Long.parseLong(timestampPart);
                long currentTime = System.currentTimeMillis();
                long oneHourAgo = currentTime - (60 * 60 * 1000);

                if (timestamp >= oneHourAgo && timestamp <= currentTime + (5 * 60 * 1000)) {
                    return true;
                }
            } catch (NumberFormatException | StringIndexOutOfBoundsException e) {
                return false;
            }
        }

        return false;
    }
}
```
