# 脑电数据分析系统 - 实时分析弹幕功能说明文档

## 1. 系统概述

本模块是基于Spring Boot框架构建的实时脑电数据分析系统，采用创新的"弹幕"展示方式，将复杂的脑电频谱分析结果以直观、实时的方式呈现给用户。系统从InfluxDB 3.2.1时序数据库中读取脑电数据，通过科学的频谱分析算法识别用户的精神状态，并通过WebSocket实时推送分析结果，为用户提供个性化的健康监测和建议。

## 2. 系统架构组件

**`analyzeSpectralData()`** - 频谱数据分析

- **科学算法：** 基于脑电频段特征的科学分析方法
- **归一化处理：** 对频谱数据进行归一化，消除个体差异
- **状态识别：** 根据各频段的相对强度识别精神状态
- **置信度评估：** 基于样本数量和频率分布评估结果可信度

```java
private AnalysisResult analyzeSpectralData(SpectralData data) {
    AnalysisResult result = new AnalysisResult();

    // 计算各频段的平均值
    Map<String, Double> averages = new HashMap<>();
    for (Map.Entry<String, List<Double>> entry : data.getBandValues().entrySet()) {
        double avg = entry.getValue().stream().mapToDouble(Double::doubleValue).average().orElse(0.0);
        averages.put(entry.getKey(), avg);
    }

    // 归一化处理
    double totalPower = averages.values().stream().mapToDouble(Double::doubleValue).sum();
    Map<String, Double> normalized = new HashMap<>();
    if (totalPower > 0) {
        for (Map.Entry<String, Double> entry : averages.entrySet()) {
            normalized.put(entry.getKey(), entry.getValue() / totalPower);
        }
    }

    result.setAverages(averages);
    result.setNormalized(normalized);

    // 分析精神状态
    analyzeMentalState(result, normalized);

    // 计算置信度
    result.setConfidenceScore(calculateConfidenceScore(data.getSampleCount(), normalized));

    return result;
}
```

## 3. 核心算法与科学依据

### 3.1 脑电频段分析

**频段定义：**

- **Delta (δ) 0.5-4Hz：** 深度睡眠、无意识状态
- **Theta (θ) 4-8Hz：** 创造性思维、深度冥想
- **Alpha (α) 8-13Hz：** 放松清醒、注意力准备
- **Beta (β) 13-30Hz：** 主动思维、专注工作
- **Gamma (γ) 30-100Hz：** 高级认知、信息整合

### 3.2 状态识别算法

**归一化处理：**

```java
// 消除个体功率差异，关注相对强度
normalized.put(band, rawValue / totalPower);
```

**状态判断逻辑：**

- Alpha主导 + 低Beta → 放松状态
- Beta主导 + 低Alpha → 专注状态
- Theta + Alpha组合 → 冥想创造状态
- Delta主导 → 困倦或深度放松
- 不均衡分布 → 状态失衡

### 3.3 置信度评估

**评估因子：**

```java
// 基于样本数量的基础置信度
double baseConfidence = Math.min((double) sampleCount / 100.0, 1.0);

// 基于频率分布均衡性的调整
double balanceScore = Math.max(0.2, 1.0 - (maxValue - 0.4));

// 最终置信度（最低30%）
return Math.max(0.3, Math.min(1.0, baseConfidence * balanceScore));
```
